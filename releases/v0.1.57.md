# Shinobi Online 0.1.57

This release introduces Combat Stamina and a redesigned battlefield interface.

## Highlights

- Added Combat Stamina (`SP`) as a third resource alongside Health (`HP`) and Chakra (`CP`).
- Running drains SP. Walking, standing idle, and resting restore it at progressively stronger rates, with recovery accelerating over time.
- Maito clan techniques now use SP instead of Chakra. Kawarimi and Strategic Retreat also consume SP.
- Holding `R` accelerates Chakra Mold. Maito shinobi cannot mold Chakra.
- Added HP/CP/SP bars inside the battlefield and an SP minibar above the player.
- Added a connected status-effect dock with compact icons and support for crowded effect sets.
- Reworked the centered action bar to seven jutsu slots and three item slots, using `1`–`7`, `T`, `G`, and `V` by default.
- Expanded battlefield view is now the default. Use `Alt+F11` to switch directly between standard and expanded layouts.
- Added battlefield-interface opacity controls and responsive scaling for smaller windows.
- Renamed the legacy green account timer to **Training Reserve** to distinguish it from Combat Stamina.

## Platform

- Includes the current Windows launcher, installer, and source-client runtime package.
- Targets Windows 10 or newer x64 with an OpenGL-capable GPU driver.
- Android downloads remain paused for this release.
- The launcher uses the public `latest.json` and `server.json` manifests for updates and Leaf endpoint discovery.
