# Shinobi Online 0.1.54

This release updates the Shinobi Online player package from the private production source.

- Includes the current player launcher.
- Includes the current Shinobi Online source-client runtime package.
- Android downloads are paused unless the operator explicitly passes Android build/APK inputs.
- Targets Windows 10 or newer x64 with an OpenGL-capable GPU driver.
- Uses the public latest.json and server.json manifests for updates and Leaf endpoint discovery.
